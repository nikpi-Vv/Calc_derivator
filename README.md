# Lab 3. Symbolic calculator

## Описание

Проект реализует символьный калькулятор на C++, который умеет:

- вычислять значение выражения (`evaluate`);
- строить символьную производную по первой переменной (`derivative`);
- вычислять значение производной в точке (`evaluate_derivative`).

Программа работает через `stdin/stdout`.

## Поддерживаемые возможности

- бинарные операции: `+`, `-`, `*`, `/`, `^`;
- унарные операции: `+`, `-`;
- вещественные числа;
- переменные;
- функции одного аргумента:
  - `sin`, `cos`, `tan`,
  - `asin`, `acos`, `atan`,
  - `exp`, `log`, `sqrt`;
- вложенные выражения и скобки;
- регистронезависимые имена переменных и функций.

## Структура проекта

- `include/` — заголовочные файлы;
- `src/` — исходный код программы;
- `tests/` — тесты;
- `external/catch2/` — простой тестовый раннер в catch-style;
- `Makefile` — сборка программы и тестов.

## Архитектура

Проект состоит из нескольких независимых модулей:

- `Lexer` — разбивает входную строку на токены;
- `Parser` — строит AST выражения;
- `Evaluator` — вычисляет значение AST;
- `Differentiator` — строит AST производной;
- `Simplifier` — упрощает выражение;
- `Printer` — печатает AST обратно в строку;
- `runApplication` — верхнеуровневая функция обработки ввода и вывода.
